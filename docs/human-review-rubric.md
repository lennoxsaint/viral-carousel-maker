# Human review rubric

Use this rubric after the automated QA report passes.

Score each item from 1 to 10:

- Scroll-stop strength: would the hook interrupt a fast feed scroll?
- Saveability: does the carousel feel worth saving?
- Clarity: can a viewer understand the point on mobile?
- Visual polish: does the design feel intentional and consistent?
- Brand fit: does it match the creator's stated style?
- CTA quality: does the final slide ask for the right action?
- Threads-native feel: does it feel like a feed-native post, not a generic deck?

Recommended release bar: average score of 8.5 or higher, with no category below 7.
