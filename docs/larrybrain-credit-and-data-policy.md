# LarryBrain Credit And Data Policy

This project borrows public growth-loop ideas from LarryBrain/Larry Marketing:

- Marketplace repo: https://github.com/OllieWazza/LarryBrain-Skill
- Marketplace: https://www.larrybrain.com/
- Public install API: https://www.larrybrain.com/api/skills/install?slug=larry-marketing&mode=files

What we adopted:

- Deep onboarding before generation
- Competitor/trend research when the user grants permission
- Reusable visual prompt anchors
- Performance tracking after publishing
- Diagnosing views, saves, clicks, and conversions differently

What we did not adopt in v1:

- TikTok posting automation
- Postiz dependency
- RevenueCat integration
- Direct Threads publishing
- Browser staging automation

Private data policy:

- Public docs may describe derived principles.
- Public docs must not include raw private corpus posts, private performance logs, API keys, tokens, or copied proprietary prompt files.
- User profile and metrics data stay local under `~/.viral-carousel-maker/`.
